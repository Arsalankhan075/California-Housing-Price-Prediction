# California-Housing-Price-Prediction
A Machine Learning project that predicts California housing prices using Multiple Linear Regression. This project includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction using the Scikit-learn California Housing dataset.

## 📌 Project Overview
This project predicts California housing prices using Multiple Linear Regression with the Scikit-learn California Housing dataset.

## 📂 Dataset
- Dataset: California Housing Dataset
- Source: Scikit-learn (`fetch_california_housing()`)

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📊 Features
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training
- Model Evaluation
- Prediction

## 📈 Model
- Multiple Linear Regression

## Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
